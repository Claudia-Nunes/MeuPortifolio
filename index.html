Projetos


1º Projeto ABP - Aplicação Web Instruções SCRUM


O 1º projeto de ABP da Fatec Jacareí visa desenvolver uma aplicação web para instruções sobre metodologias ágeis, com foco no SCRUM na área de TI. A aplicação apresentará conceitos, papéis e eventos do SCRUM de forma interativa, utilizando HTML, CSS e JavaScript para a construção da interface e funcionalidades. O projeto busca facilitar o aprendizado e aplicação da metodologia, além de desenvolver habilidades técnicas e colaborativas dos alunos.



GitHub Repository

2º Projeto ABP - Aplicação Web para Aluguel de Bicicletas Peer-to-Peer


O 2º projeto de ABP da Fatec Jacareí consiste no desenvolvimento de uma aplicação web para aluguel de bicicletas no modelo Peer-to-Peer. A plataforma conecta locadores e locatários, permitindo que ciclistas compartilhem suas bicicletas com outros entusiastas. A aplicação utiliza TypeScript, PostgreSQL e React, garantindo uma interface moderna, segura e escalável para facilitar a experiência dos usuários no aluguel e gerenciamento das bicicletas.



GitHub Repository

3º Projeto ABP - Sistema Web para Gerenciamento de Projetos


O 3º projeto de ABP da Fatec Jacareí tem como objetivo o desenvolvimento de um Sistema Web para gerenciamento de projetos, oferecendo funcionalidades para controle de tarefas, acompanhamento de progresso e análise de dados. A aplicação utiliza MongoDB, TypeScript e React, proporcionando uma solução escalável e eficiente para gerenciar projetos de diferentes áreas, com visualizações gráficas e métricas detalhadas para melhor tomada de decisão.



GitHub Repository

4º Projeto ABP - Aplicação Mobile para Controle de Alimentação e Nutrição


O 4º projeto de ABP da Fatec Jacareí tem como objetivo desenvolver uma aplicação mobile que permite aos usuários registrar e monitorar sua ingestão diária de calorias e nutrientes, auxiliando no controle da alimentação e na manutenção da saúde. A aplicação utiliza TypeScript, React Native, MongoDB e PostgreSQL, proporcionando uma interface intuitiva para acompanhamento de macros (carboidratos, proteínas, gorduras) e consumo de água, oferecendo insights para uma alimentação equilibrada.
